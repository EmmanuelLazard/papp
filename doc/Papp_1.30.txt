Dear all,I am proud to announce the release of PAPP 1.30.This new version, available on http://www.ffothello.org/info/appa.php, shows major improvements and is a recommended upgrade for all users.The main changes are the following:- The 'nouveaux' file (which records new players and nationality changes) is automatically added to the '__papp__' file and the ELO file as comments. This enables me to process more easily these files.- The Brightwell constant is no longer restricted to integers.- When launching Papp for a new tournament, the program asks for the tournament name, the number of rounds and the Brightwell constant. These informations are put in the __papp__ file and ARE NO LONGER READ from the 'papp.cfg' file. This enables Papp to have only one papp.cfg file which can be left unchanged by the user whatever tournament is run (round-robin, swiss...). If the papp.cfg is not available, Papp will issue a warning and generate a standard papp.cfg file (which should be ok for 99,9% users). If Papp is run with a __papp__ file available, it will of course use the data provided in the file and will not ask the user.  This is backward compatible: if a former papp.cfg is found (<= 1.29), Papp will issue warning and will disregard the specified number of rounds and Brightwell constant before asking the user for the values if they cannot be found in a saved __papp__ file; if a former __papp__ is found (without the informations), Papp will issue a warning and ask the user before adding them to the __papp__ file. These new __papp__ files CAN be read with former (<= 1.29) versions (but without using the informations of course).- A new command is introduced in the papp.cfg file which tells Papp to save all pairings/results/standings/crosstables files in a directory named after the tournament. This is ENABLED by default, so look for this directory when looking for those files. A backup copy of the __papp__ file is also saved inside.- The documentation has been updated (unfortunately still only in French).Please send me feedback, remarks, critics, suggestions...Emmanuel**************************************************************Bonjour ˆ tous,J'ai le plaisir de vous annoncer la sortie de PAPP 1.30.Cette nouvelle version, disponible sur http://www.ffothello.org/info/appa.php, inclut de nombreuses amŽliorations et est recommandŽe pour tous les utilisateurs.Les changements principaux sont les suivants :- Le fichier 'nouveaux' (qui contient les nouveaux joueurs ainsi que les changements de nationalitŽ) est automatiquement ajoutŽ aux fichiers __papp__ et ELO sous forme de commentaires.
